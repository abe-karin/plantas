# Guia de Estudo - Plantas Indicadoras e Qualidade do Solo
Este documento serve como um registro estruturado dos conceitos, ferramentas e métodos explorados para compreender como a vegetação espontânea atua como um biosensor da saúde do solo.
## 1. Objetivos do Estudo
Diagnóstico Natural: Utilizar a presença de "ecótipos" para identificar condições químicas (pH e nutrientes) e físicas (compactação) do solo sem depender exclusivamente de análises laboratoriais
.
Manejo Sustentável: Compreender a transição do controle de "plantas daninhas" para a correção da causa raiz no solo
.
Valorização do Saber Local: Integrar a percepção empírica do agricultor com dados técnico-científicos
.
Subsolagem Biológica: Explorar o uso de raízes profundas para recuperação da estrutura física do terreno
.

## 4. Miniguia de Referência Rápida
Use este guia para um diagnóstico visual imediato no campo:

| Se você vir... | O solo provavelmente está... | Ação recomendada |
| :--- | :--- | :--- |
| **Beldroega / Dente-de-leão** | Fértil, boa drenagem e rico em Boro | Manter o manejo; solo em boas condições. |
| **Amendoim-bravo / Barba-de-bode** | Ácido (pH baixo), pobre em P, K e Ca | Realizar calagem e adubação mineral. |
| **Guanxuma / Bardana / Tiririca** | Altamente compactado e/ou mal drenado | Aerar o solo; usar plantas de subsolagem (ex: feijão-guandu). |
| **Urtiga** | Rico em Nitrogênio, mas pobre em Cobre | Equilibrar micronutrientes. |
| **Capim-carrapicho** | Compactado e sofrendo erosão superficial | Recuperar a cobertura vegetal e evitar solo exposto. |

## 5. Ferramentas de Aprendizagem Criadas

Plantas Quiz: Teste de conhecimentos para fixação das espécies e suas funções.
Glossário: Definições de termos como ecótipos, bioporos e filtros ambientais.
